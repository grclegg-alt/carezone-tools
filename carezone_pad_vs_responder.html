Static PAD or cardiac responder?
A ward-by-ward recommendation across all 32 Scottish councils

Each ward is scored across four factors — response time, population density, arrest incidence and existing PAD use — to indicate whether more static defibrillators or a community cardiac responder scheme is the better investment of CARe Zone effort.

⚠ Population density is estimated from response time patterns — not real data. All other inputs are real SAS data. This tool is for directional planning only.

Real data (SAS council data cut): cardiac arrests per ward, PAD use rate, bystander CPR rate, median ambulance response time.   Estimated/synthesised: population density — derived algorithmically from response time and arrest patterns as a proxy; not based on census data. The recommendation algorithm weights four factors: population density (35%), response time (35%), arrest count (15%), and existing CPR/PAD behaviour (15%). This tool is for exploratory planning conversations only — it does not constitute clinical or operational advice. Signal strength reflects the confidence margin between PAD and Responder scores.
